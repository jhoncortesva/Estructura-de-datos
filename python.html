Grafos

Un grafo es un conjunto de objetos llamados “nodos” unidos por enlaces llamados “aristas” o “arcos” que permiten representar relaciones binarias entre elementos de un conjunto. Si

tiene flechas se llama grafo dirigido o digrafo. Si no tiene flechas se llama grafo no dirigido.



Python no tiene definido un tipo de dato para Grafos. Para poder utilizarlos, se puede utilizar un módulo como ‘networkx’ o se pueden implementar manualmente.
